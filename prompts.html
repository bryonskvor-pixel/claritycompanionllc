These are Claude Project configurations built from real conversations about real human needs. Paste one into your Claude Project instructions and unlock a purpose-built experience in about 60 seconds. No app. No subscription. No paywall.

How It Works

One paste. One project.
A completely different Claude.

Claude Projects let you configure a persistent identity that runs silently in every conversation inside that project. You never have to set it up again. Just open the project and start talking.

1
Create a Project

In Claude.ai, create a new Project. Name it whatever feels right. This becomes your dedicated space.

2
Paste the Prompt

Open Project Settings and paste the full prompt text into the Instructions field. Save it.

3
Start a Session

Open a new conversation inside the project. The configuration runs automatically. Just say what you came to say.
